# CO₂-Footprint Webanwendung

Diese Webanwendung ist im Rahmen einer Fallstudie zur Visualisierung fiktiver CO₂-Emissionsdaten entstanden.
Der Fokus liegt auf der konzeptionellen und technischen Umsetzung einer interaktiven, responsiven Weboberfläche.

## Technologien
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Vue.js (clientseitig, ohne Build-Tool)

## Projekt starten
Die Anwendung ist vollständig clientseitig umgesetzt.

1. Repository herunterladen oder klonen
2. `index.html` im Browser öffnen

## Dokumentation
Die vollständige wissenschaftliche Dokumentation (Umsetzungskonzeption) ist als PDF eingereicht und
nicht Bestandteil dieses Repositories.
